# Project 1: Flash Card Webpage

The flash card webpage is the first project of the triple10 AI Assisted - SE program.
It has been created using HTML, CSS, and JS based on a series of design briefings.

## Project Features

- Deck gallery with interactive delete buttons and navigation links.
- Card gallery within individual decks with interactive delete, flip, and practice buttons.
- Card carousel.
- Mobile compatibility with responsive display queries.

## Technologies Used

- VS Code
- GitHub
- GitBash
- Copilot
